# persistent_ents

Данная библиотека позволяет создавать persistent entity, которые будут оставаться в том же положении после перезагрузки сервера, а так же предоставляет API для назначения им синхронизируемых свойств (по аналогии с стандартными NW функциями), значения которых сохраняются при перезагрузке сервера.
